# Input

Componente atomico para entrada de texto. Ele suporta tokens de tema, tamanhos, estados de erro, estado disabled, helper text e icones laterais.

## Quando Usar

- Campos simples de texto
- Entradas com icones contextuais
- Campos controlados ou nao controlados pelo consumidor

## Props Principais

| Prop | Tipo | Descricao |
| --- | --- | --- |
| `size` | `sm`, `md`, `lg`, `xl` | Define altura e escala do campo |
| `error` | `boolean` | Aplica estado visual de erro |
| `helperText` | `string` | Texto de apoio abaixo do campo |
| `iconLeft` / `iconRight` | `ReactNode` | Elementos laterais |
| `disabled` | `boolean` | Bloqueia interacao |

## Exemplo

```tsx
<Input
  placeholder="nome@email.com"
  helperText="Use um email valido."
  iconLeft="@"
/>
```
